== Description ==

This plugin track records of wordpress user login with set of multiple information like ip, date , time, country , city, user name etc.


### Plugin Features :

* ability to filter by user name, successful/failed logins, month and year
* export into CSV file
* Track ip ;
* user country info
* user city info
* Browser info

Plugin by <a href="http://weblizar.com">Weblizar</a>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `User-login-log` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Then Go to user menu in WordPress' admin panel.
5. final view your log.
